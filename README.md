# Leo Thaylor | Portfólio BJJ

Site estático one-page para o portfólio pessoal de Leo Thaylor.

## Estrutura

- `index.html`: conteúdo e seções do site.
- `styles.css`: identidade visual, responsividade e layout.
- `script.js`: menu mobile e estado do cabeçalho.
- `assets/images/hero-bjj-placeholder.png`: imagem placeholder do hero.

## Publicação

Este projeto pode ser publicado diretamente pelo GitHub Pages usando a branch principal e a pasta raiz.
